# PPI-Network-Analysis

This repository contains the implementation of our proposed framework for comparative analysis of protein–protein interaction networks (PPINs) of SARS-CoV-2 and H1N1 influenza.

The study investigates structural similarities and differences between the two viral interaction networks using advanced network science and machine learning techniques.

---

## Study Objectives

- Comparative analysis of SARS-CoV-2 and H1N1 PPINs  
- Investigation of complex network properties:
  - Graph density  
  - Degree distribution  
  - 21 centrality measures  
- Fitting statistical distributions (power-law and exponential models)  
- Dimensionality reduction  
- Application of unsupervised learning methods to identify the most informative centrality measures

## Requirements

- R version 1.1.453 or higher  
- R package: `igraph`

---

## Methodological Overview

The framework integrates:

- Network topology analysis  
- Statistical distribution modeling  
- Feature extraction  
- Dimensionality reduction techniques  
- Unsupervised machine learning methods  

This approach enables systematic comparison of viral PPIN structures and identification of key topological characteristics.
